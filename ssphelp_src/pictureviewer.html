画像ビューワ

画像ファイルを表示し、拡大縮小・回転・トリミングなどの簡単な操作や、形式を変えて保存・コピーができるビューワウィンドウです。
SSPが対応している多くの画像形式をその場で確認できるほか、複数の画像をタブで並べて切り替えながら閲覧できます。

シェルやバルーンの素材画像、撮影した写真など、ちょっとした画像を確認したり形式変換したりするのに便利です。

対応する画像形式

対応する主な画像形式は以下の通りです。
「対応状況」のSSP全域はSSP本体が標準で読み込める形式（シェルやバルーンの素材などにも使える）、ビューワのみはこの画像ビューワでのみ表示できる形式（OSのWICコーデック経由）を表します。

形式 拡張子 対応状況 注釈
PNG / APNG .png / .apng SSP全域 アルファ（透明）対応。同名の.pnaファイルによる外部アルファ併用可。APNG（アニメーションPNG）にも対応。
WebP .webp SSP全域 アニメーション・透過対応。
GIF .gif SSP全域 アニメーション・透過対応。
MAG .mag SSP全域 レトロ形式（MAKICHAN GRAPHICS、PC-9801時代）。
Pi .pi SSP全域 レトロ形式（PC-9801時代のPi形式）。
JPEG .jpg / .jpeg SSP全域 EXIF Orientation（撮影時の向き情報）を自動適用します。
BMP .bmp SSP全域 Windowsビットマップ。
TIFF .tif / .tiff SSP全域 マルチページ対応（前のページ／次のページで切り替え）。
X BitMap / X PixMap .xbm / .xpm SSP全域 X Window系のテキスト形式画像。
TGA .tga SSP全域 Truevision Targa。透過対応。
Netpbm（Portable anymap） .pbm / .pgm / .ppm / .pam / .pnm / .pfm SSP全域 ポータブルビット／グレイ／ピックスマップ系。.pfmは浮動小数点（HDR）形式。
PCX .pcx SSP全域 ZSoft Paintbrush。
QOI .qoi SSP全域 Quite OK Image Format。
ICO .ico SSP全域 Windowsアイコン形式。
PSD .psd SSP全域 Adobe Photoshop。統合済み画像に加え、各レイヤーも個別に表示可（前のページ／次のページで切り替え）。
PSP .psp / .pspimage SSP全域 Paint Shop Pro。version4.0以降。統合済み画像に加え、各レイヤーも個別に表示可（前のページ／次のページで切り替え）。
OpenRaster .ora SSP全域 統合プレビュー（mergedimage.png）を表示します。
Krita .kra SSP全域 統合プレビュー（mergedimage.png）を表示します。
CLIP STUDIO .clip SSP全域 最初のページはキャンバスプレビュー画像（全体の縮小プレビュー）を表示、2ページ目以降は各レイヤーの個別表示（前のページ／次のページで切り替え）。
HEIF / HEIC .heif / .heic ビューワのみ OSの「HEIF 画像拡張機能」が必要です。
AVIF .avif ビューワのみ OSの「AV1 ビデオ拡張機能」が必要です。
JPEG XR .jxr / .wdp / .hdp ビューワのみ HD Photo。
DDS .dds ビューワのみ DirectDraw Surface。先頭フレームのみ表示します。
カメラRAW .cr2 / .nef / .arw など ビューワのみ 各社のRAW形式。OSの「RAW 画像拡張機能」に依存します。

「ビューワのみ」の形式は、OSにインストールされたコーデック（画像拡張機能）に依存します。上表は代表例で、環境によっては読めない場合があります。
また「ビューワのみ」経路はアニメーション・マルチページに対応せず、常に先頭の1コマのみ表示されます。

起動方法

画像ビューワは、次のいずれかの方法で開けます。

右クリックメニュー →「機能」→「画像ビューワ」
画像を読み込んでいない空の状態でビューワを起動します。
⇒「機能」のページも参照してください。
画像ファイルをゴーストにドラッグ＆ドロップ
画像ファイルをゴーストの上にドロップすると、ゴースト側がそのファイルに反応しなかった場合に、自動的にこのビューワで開きます。
画像として読み込めないファイルの場合はビューワは起動しません。

起動後は、ビューワ上で「画像を開く」やクリップボードからの貼り付け、ビューワ内へのドラッグ＆ドロップで、さらに画像をタブとして追加できます。

画面の構成

メニューバー
「ファイル」「編集」「表示」の各メニューから、すべての操作を行えます（詳細は後述）。
ツールバー
よく使う操作をボタンで並べたものです。各メニュー項目に対応しており、ボタンにマウスを乗せると機能名（ツールチップ）が表示されます。
トリミング・左上の色を透明化は、ONのときボタンが押し込まれた表示になります。背景の切り替えボタンは、押すたびにアイコンが現在の背景（市松／白／黒）に変わります。
タブ
開いている画像ごとにタブが並びます。タブをクリックすると表示する画像を切り替えられます。
画像表示領域
画像を表示する領域です。画像が領域より大きい場合はスクロールバーが表示されます。
ステータスバー
ウィンドウ下部に、現在の画像の「寸法（幅×高さ）」「拡大率」「回転角度（左右反転中は <|> を併記）」「形式」が表示されます。
トリミング中は、左側に選択している範囲の座標が表示されます。

上部メニュー

ファイル

画像を開く...[Ctrl+O]
ファイル選択ダイアログを開き、選んだ画像を新しいタブで開きます。
再読み込み[Ctrl+R / F5]
現在表示している画像を、ディスクから読み込み直します。
ファイルから開いた画像は外部での更新を自動的に検知して再読み込みしますが、手動で読み直したいときに使います。
名前を付けて保存...[Ctrl+S]
表示中の画像をPNG／BMP／JPEG形式で保存します。
回転・左右反転・トリミング・サイズ変更を行っている場合は、その表示状態（見たままの内容）で保存されます。

編集

コピー[Ctrl+C]
表示中の画像を、透明度を保持したままクリップボードへコピーします。
回転・反転・トリミング中はその表示状態でコピーされます。
貼り付け[Ctrl+V]
クリップボードにある画像を、新しいタブとして取り込みます。
サイズを変更...
ダイアログで幅・高さを指定し、画像を拡大縮小します。
元画像は保持されており、元と同じ寸法を指定すると元の画像に戻ります（非破壊）。
トリミング
トリミング（切り抜き）モードのON／OFFを切り替えます。
ONにすると、画像上をマウスでドラッグして切り抜く範囲を指定できます。保存・コピーの際に、その範囲だけが対象になります。

表示

実際のサイズ[Ctrl+0]
拡大率を100%（等倍）に戻します。
ズームイン[Ctrl++] / ズームアウト[Ctrl+-]
表示を1.25倍ずつ拡大・縮小します。拡大率は5%〜1600%の範囲です。
ウインドウに合わせる
画像全体がウィンドウに収まるよう、自動的に拡大率を調整します。ウィンドウの大きさを変えると追従します。
拡大率を指定...
ダイアログで拡大率（％）を直接入力して表示します。
前のページ[PageUp] / 次のページ[PageDown]
マルチページTIFFやアニメーション画像で、前後のコマ（ページ）へ移動します。
左回転 / 右回転
表示を90度ずつ左・右に回転します。表示上の回転であり、元の画像データは変化しません（保存・コピーには反映されます）。
左右反転
表示を左右に反転（鏡像）します。こちらも表示上の操作で、元の画像データは変化しません。
黒背景 / 白背景 / 市松模様背景
透明部分の背後に敷く背景を切り替えます（初期値は市松模様）。透明部分の見え方を確認するのに使います。
この設定はすべてのタブで共通で、次回起動時にも引き継がれます。
左上の色を透明化
画像の左上（座標0,0）の色と同じ色を、表示上だけ透明として扱います。背景に透明色を持たない画像の透け具合を確認するのに便利です。
この設定はすべてのタブで共通で、次回起動時にも引き継がれます。

「左上の色を透明化」は、もともと透明度（アルファ）を持たない画像にのみ働きます。すでに透明部分を持つ画像では、意図しない部分まで消えてしまうのを防ぐため、設定がONでも適用されません。

マウス・キーボード操作

マウスホイールを回すと、カーソルの位置を中心に拡大・縮小できます。
タブは、中クリックまたはタブ上の「×」ボタンで閉じられます。タブを右クリックして「閉じる」を選んでも閉じられます。
トリミングモード中は、画像上をドラッグして切り抜く範囲を指定します。

主なキーボードショートカットは以下の通りです。

キー 機能

Ctrl+R / F5再読み込み

Ctrl+V貼り付け

各機能の詳細

拡大・縮小

拡大率は5%〜1600%の範囲で、ズームイン／アウトは1.25倍ずつ変化します。
マウスホイールではカーソル位置を中心に拡大縮小されるため、見たい場所を素早く拡大できます。「ウインドウに合わせる」を選ぶと、画像全体が常にウィンドウに収まるよう自動調整されます。

回転・左右反転

左回転・右回転（90度単位）と左右反転ができます。これらは表示上の操作で、元の画像データそのものは加工されません。

回転・反転した状態で「名前を付けて保存」や「コピー」を行うと、その見たままの向きで出力されます。
なお、ページを送ったりサイズを変更したりすると、回転・反転・トリミングの状態は解除されます。

背景・透明部分の表示

透明部分の背後に敷く背景を、黒・白・市松模様から選べます（初期値は市松模様）。市松模様は透明部分が分かりやすいよう市松状のタイルで表示します。
「左上の色を透明化」と組み合わせると、透明色を持たない画像の透け方を確認できます。

サイズを変更・トリミング

「サイズを変更」では幅・高さを指定して拡大縮小できます。元画像は保持されるため、元の寸法に戻すこともできます。
「トリミング」をONにして画像上をドラッグすると、切り抜く範囲を指定できます。指定した範囲は保存・コピーの対象になり、ステータスバー左側に座標が表示されます。トリミングはタブごとの設定で、元画像は変化しません。

保存・コピー

「名前を付けて保存」ではPNG／BMP／JPEGを選べます。元がJPEGで、サイズ変更・トリミングをしていない場合は、再エンコードせずに保存します。このとき回転・反転は画質を劣化させずに反映され（無劣化回転）、EXIFなどのメタ情報は削除されます。
「コピー」では透明度を保持したままクリップボードへ送るため、対応するアプリにそのまま透明部分付きで貼り付けられます。

自動再読み込み

ファイルから開いた画像は、外部のソフトでそのファイルが更新されると自動的に検知し、約1秒後に表示を更新します。画像を編集しながらビューワで確認する用途に便利です。

マルチページ・アニメーション

マルチページTIFFや、GIF・APNG・WebPなどのアニメーション画像は、「前のページ」「次のページ」でコマを切り替えて確認できます（自動再生は行いません）。
PSDは各レイヤーも「前のページ」「次のページ」で切り替えて個別に確認できます。
